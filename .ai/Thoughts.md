 This project, `amebic`, will be something I'm building that essentially is going to be like ReMotion for still graphics. So what you'll be able to do is to find one document or composition in React, and then you can get out a bunch of still graphics in different formats that dynamically generate and so on. And we'll have a cool little preview web UI and then the ability to statically render. We should support transparency, all the normal cool, like graphical effects that could be powered by WebGL or CSS and everything along those lines.
